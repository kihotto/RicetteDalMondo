# 🌍 Ricette dal Mondo

Applicazione mobile per esplorare la cultura culinaria globale.  
Cerca un paese tramite il mappamondo 3D e scopri ricette autentiche con ingredienti, dosi e preparazione.

---

## ✨ Features Principali

- 🌐 Mappa globale 3D interattiva
- 📖 Catalogo ricette per paese
- 🍽️ Dettagli ricetta con ingredienti e preparazione
- 📱 Interfaccia responsive e intuitiva
- ☕ Integrazione con backend Java/Spring Boot

---

## 🛠️ Stack Tecnologico

### Frontend

| Libreria                | Versione | Descrizione                             |
| ----------------------- | -------- | --------------------------------------- |
| React Native            | 0.81.5   | Framework mobile cross-platform         |
| Expo                    | 54.0.32  | Toolchain con accesso a API native      |
| Expo Router             | 6.0.22   | Navigazione file-based                  |
| NativeWind              | latest   | Tailwind CSS per React Native           |
| React Three Fiber       | 9.5.0    | Binding React per Three.js              |
| Three.js                | 0.182.0  | Rendering 3D WebGL per la mappa globale |
| React Native Reanimated | 4.1.1    | Animazioni ad alte prestazioni          |
| TypeScript              | 5.9.2    | Type safety e developer experience      |

### Backend

| Tecnologia  | Descrizione                 |
| ----------- | --------------------------- |
| Java        | Linguaggio backend          |
| Spring Boot | Framework REST API          |
| MongoDB     | Database NoSQL              |
| Render      | Piattaforma di deploy cloud |

> Il backend è gestito in un repository dedicato (frontend/backend separation).
> Per dare un occhiata al Backend https://github.com/LLeo93/recipe

### Dev Tools

- Metro Bundler — Bundler React Native
- Tailwind CSS — Utility-first styling
